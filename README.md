# Max0n accounts template

![Latest Stable Version](/preview.gif)

Для запуска:
- Скачать и установить [nodejs](https://nodejs.org/)
- `npm install -g @angular/cli`
- `npm install`
- `npm start`

## DevelopmentRun
- `npm start`

## Builds
**DEV**
- `npm run build-dev`

**PROD**
- `npm run build-prod`
